# React Multistep Form

A beautiful and modern multistep evaluation form built with React and Vite.

## Project Structure

- `multistep_react_form/`: The main React application.
- `multistep_react_form/cypress/`: End-to-End tests.

## Getting Started

To get started with the project, navigate to the `multistep_react_form` directory and install the dependencies:

```bash
cd multistep_react_form
npm install
```

For more details on how to run and test the application, see the [Project README](file:///c:/Lucas%20Orsi/react_multistep_form/multistep_react_form/README.md).
